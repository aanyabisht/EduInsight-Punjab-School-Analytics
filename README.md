# EduInsight Punjab: School Analytics 

## Project Overview

This project presents a comprehensive data analysis of a Punjab government primary school based on survey-collected data. The objective is to uncover meaningful insights related to student demographics, gender distribution, category representation, and participation in extracurricular activities such as sports and school events.

The project follows a complete data analytics pipeline, transforming raw survey data into actionable insights through structured analysis and interactive visualization.

---

##  Objectives

* Analyze class-wise gender distribution and identify imbalances
* Evaluate category representation (SC/OBC vs General)
* Assess student participation in sports (Kabaddi, Kho-Kho)
* Examine engagement in school events (Annual Day, Sports Day)
* Provide insights to support better educational and participation outcomes

---
##  Data Source
The dataset used in this project was collected through a field survey conducted by me as part of an academic opportunity provided by my college. This primary data collection adds authenticity to the analysis and reflects real-world conditions of a Punjab government primary school.

##  Tools & Technologies

* **Excel** – Data cleaning and initial structuring
* **Python (Pandas, Matplotlib, Seaborn)** – Exploratory Data Analysis and visualization
* **SQL** – Data querying and validation
* **Tableau** – Interactive dashboard creation

---

##  Workflow

1. **Data Collection**

   * Conducted a survey in a Punjab government school (Nursery to 5th class)

2. **Data Cleaning (Excel & Python)**

   * Structured raw data
   * Handled inconsistencies and created derived metrics (Total, Percentages, Gender Gap)

3. **Exploratory Data Analysis (Python)**

   * Visualized gender distribution, participation trends, and class-wise patterns

4. **SQL Analysis**

   * Created structured tables
   * Performed queries to validate insights and calculate metrics

5. **Dashboard Development (Tableau)**

   * Built an interactive dashboard highlighting key insights

---

##  Key Insights

* Female participation is higher in lower classes but declines in higher grades
* Noticeable gender gaps exist in specific classes, indicating imbalance
* Majority of students belong to SC/OBC category
* Sports participation is concentrated in higher primary classes (4th–5th)
* Active involvement in school events reflects strong extracurricular engagement

---

##  Dashboard

The Tableau dashboard provides a visual summary of:

* Gender distribution across classes
* Category-wise student distribution
* Sports participation (Kabaddi & Kho-Kho)
* Event participation trends


##  Recommendations

* Encourage female participation in higher classes
* Expand sports opportunities to lower grades
* Continue support for SC/OBC students through inclusive policies
* Enhance engagement through structured extracurricular programs
